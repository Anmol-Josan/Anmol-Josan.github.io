Welcome to Web 2!

Today, we will mess with light and dark modes.

First, lets dive into changing font colors

In CSS and JS we can make a button change a variable by using this:

    const setTheme = theme => document.documentElement.className = theme;

Then we create two buttons that look like this:

            <button class="dark theme-switcher bigger-text" onclick="setTheme('dark')">Dark
<button class="light theme-switcher bigger-text" onclick="setTheme('light')">Light

In our CSS we put:

            :root,
:root.light {
    --bg-color: #000000;
    --container-color: #ffffff;
    --text-color: rgb(0, 0, 0);
}

:root.dark {
    --bg-color: #565656;
    --container-color: #000000;
    --text-color: #ffffff;
}

Later, wherever we want to change the color, so for example, the background of the container, we put this and change the --container-color to the corresponding :

            background: var(--container-color);

And that's it! You can now switch text light and dark modes.
