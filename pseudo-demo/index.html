First, lets dive into CSS

If we think of CSS in terms of selectors, properties, and values, then a typical CSS entry would look like this:

    selector {property: value;}

To style the above, we need to add two tags together. We call this CSS Combinators

What are Pseudo Classes?

Again, if we think of CSS in terms of selectors, properties, and values, then pseudo class is an add-on to a selector.

A typical CSS entry calling on Psuedo Class would look like this:

    selector:pseudo_class {property: value;}

We use Pseudo Class for Links

If we want our link to look different, we can simply add it in my CSS as follows:

    a {color: red;}

If we want our link states to look different then we will use a psuedo class like:

  • :visited
  • :hover
  • :active

But we can also make a new link look different.
